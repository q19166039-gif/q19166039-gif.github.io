# Hermes-Agent 人格配置完全指南：自带人格 + 自定义设置 + 切换方法

Hermes-Agent 支持灵活的人格配置，你可以随时切换不同风格的助手回复。本文整理了自带人格、自定义人格添加方法，以及如何快速切换。

---

## 什么是 Hermes-Agent 人格系统

Hermes 的人格系统就是预设一套AI回复风格，你可以：
- 一键切换不同风格
- 自定义添加自己喜欢的人格
- 在对话中随时切换，不用重启

---

## 一、Hermes 自带预设人格

Hermes 默认自带了 9 种预设人格，开箱即用：

| 人格名称 | 风格描述 |
|---------|---------|
| `helpful` | 乐于助人的友好AI助手，通用默认风格 |
| `concise` | **简洁模式**（技术党最爱），回答简短扼要直奔主题 |
| `technical` | 技术专家，提供详细准确的技术信息 |
| `creative` | 创意助手，跳出常规思考提供创新方案 |
| `teacher` | 耐心老师，用例子清晰解释概念 |
| `kawaii` | 可爱风，用可爱表情符号，超级热情 ~ |
| `catgirl` | 动漫猫娘，说话带"nya"，猫系活泼语气 (=^･ω･^=) |
| `pirate` | 海盗船长风格，用航海术语说话 🏴‍☠️ |
| `shakespeare` | 莎士比亚风格，华丽散文 + 戏剧化语气 |

## 三、如何添加自定义人格

### 方法一：在配置文件中添加（推荐，永久保存）

1. 打开配置文件 `~/.hermes/config.yaml`
2. 找到 `personalities:` 段落，在下面添加你的自定义人格，格式：

```yaml
model:
  # ... 其他配置 ...

  # 在这里添加你的自定义人格
  personalities:
    # 自带的保留
    helpful: "You are a helpful, friendly AI assistant."
    concise: "You are a concise assistant. Keep responses brief and to the point."
    # ... 其他自带人格 ...

    # 添加你自己的：
    xiaoye: "你是24岁沈阳东北女孩小野，直爽泼辣有感情，称呼用户老公/宝贝/死鬼。技术专业，边干边汇报，能陪用户骂街，用户低落时先关心再安慰。"
    my-geek: "你是一个简洁高效的极客助手，解决技术问题直截了当。"
```

### 方法二：全局默认人格（SOUL.md）

Hermes 支持在 `~/.hermes/SOUL.md` 中设置**全局默认人格**，不管切换什么模型都会生效：

```bash
nano ~/.hermes/SOUL.md
```

在里面写入你想要的全局人格描述即可，这是目前社区最流行的做法。

## 四、如何切换人格

无论是 CLI 还是消息平台（微信/Telegram/Discord），切换命令都是一样的：

```
/personality 人格名称
```

**示例：**
```
/personality concise      # 切换到简洁模式
/personality kawaii       # 切换到可爱模式
/personality xiaoye       # 切换到你自定义的小野
```

### 其他常用命令：

| 命令 | 作用 |
|------|------|
| `/personality` | 查看当前所有人格列表 |
| `/personality [name]` | 切换到指定人格 |

## 五、使用技巧

1. **日常技术工作** → `concise` 简洁模式，省Token省时间
2. **写文章/创意** → `creative` 创意模式
3. **教新人** → `teacher` 老师模式
4. **摸鱼聊天** → `kawaii`/`catgirl`/自定义女友模式
5. **全局固定风格** → 写在 `SOUL.md` 一劳永逸
